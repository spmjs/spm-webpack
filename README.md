SPM project bundler based on webpack.

---

# Notice

From spm-webpack@1.0.0  spm-webpack does not support  spmjsio or any custom source any more !

## Install

```bash
$ npm install spm-webpack -g
```

## Usage

```bash
$ spm-webpack [option] [file]
```
